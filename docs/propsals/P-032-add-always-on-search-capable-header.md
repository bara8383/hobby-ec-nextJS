# P-032: Add Always On Search Capable Header

- Category: concern
- Priority: high
- Status: pending
- Related: 

---

## 概要
Amazon/楽天のような「常時検索＋購買補助ヘッダー」と比較すると、現行の `components/layout/SiteHeader.tsx` はリンクナビ中心で、検索ボックス・カテゴリドロップダウン・カート件数・セール導線が未実装。トップ導線は `app/page.tsx` のヒーロー内検索に依存しており、下層回遊時の再検索効率が低い。【対応案】ヘッダーに常設検索コンポーネントを追加するか、既存 `/search` への入力UIを共通化するかを決定し、カテゴリ切替・カート件数表示・販促導線の採用範囲（常設/季節限定）を仕様化する。
